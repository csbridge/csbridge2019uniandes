Hoja de Sintaxis de JavaScript

Declarar y usar variables

Las variables en JavaScript no tienen un tipo fijo como en Java. Debido a esto, no tiene que especificar el tipo al crear una variable. En su lugar, utilizará la palabra clave var. 

var a = 10; var b = 20; var c = a + b;

Si las declaraciones

Si las declaraciones son exactamente las mismas que en Java. 

if (algunaFunciónBooleana()) { return 1; }

Ciclos while

Ciclos while que son los mismos también. 

while (algunaCondicion()) { suma += 1; }

Ciclos for

Los ciclos for son casi exactamente iguales, excepto la variable del contador. 

// Encuentra la suma de los números del 0 al 10, ambos inclusive. var suma = 0; for (var i = 0; i <= 10; i++) { suma += i; }

Funciones

Observe dónde está acostumbrado a ver el vacío privado, utilizará la función de palabra clave. Observe que los parámetros no tienen el tipo especificado. 

función algunaFunción (a, b) { return a === b; }

Los operadores

Todos los operadores son iguales, excepto la comparación (==). El operador de comparación en Java es == (dos signos iguales), mientras que la comparación en JavaScript es ===. 

a === b; // a es igual ab a !== b; // a no es igual ab

Oyentes del evento 

objeto.evento = función(e) { // tu codigo }

Eventos comunes

  • al hacer clic
  • Onmousedown
  • onmouseup
  • onkeypress
  • onkeydown
  • onkeyup
