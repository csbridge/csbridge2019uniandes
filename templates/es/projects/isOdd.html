Esta demostración muestra un método simple que toma la entrada y devuelve si esa entrada es impar o no.

Solución 

 clase pública EsImpar extiende ProgramaDeConsola {

	public void run () {
		para (int i = 0; i <100; i ++) {
			imprimir (esImpar (0));
		}
	}

	booleano privado esImpar (int valor) {
		retorno (valor% 2) == 1;
	}
}
