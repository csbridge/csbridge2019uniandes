Método promedio

Escrito por Chris Piech y Mehran Sahami.

Esta demostración muestra un método simple que toma dos entradas y devuelve el promedio:

private double promedio(double a, double b)

Solución 

 clase pública MétodoPromedio extiende ProgramaDeConsola {

	public void run () {
		imprimir (promedio (4, 5));
		imprimir (promedio (10, 2));
		imprimir (promedio (-4.1, 9.8));
	}

	Promedio doble privado (doble a, doble b) {
		retorno (a + b) / 2;
	}
}
