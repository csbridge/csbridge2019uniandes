Agregar y quitar de la ventana

Cuando creas objetos de tipo SObjeto, necesitas agregarlos a la ventana. Despues de crear los objetos con sus constructores, necesitas agregarlos para que se aparezcan en la ventana. Para eso, se usa estos métodos, sin ubicación o con ubicación respectivamente.
agregar(objeto) // en la ubicación que indica el objeto
agregar(objeto, x, y) // en la ubicación pasado
También puedes quitar un objeto o quitar todo de la pantalla.
quitar(objeto) // quita este objeto
quitarTodo() // quita todos los objetos que ya agregaste
Esos otros métodos de EsGraphics te puede servir también:
pausa(millisegundos) // pausa el código
GObjeto darObjetoA(x, y) // retorna el objeto en esta ubicacion

Constructores

Todos los objetos de la librería gráfica son tipos de SObjeto, y los constructores están mostrado aquí. Los objetos se puede construir sin ubicación o con ubicación respectivamente. En el constructor o cuando lo agregues, necesitas dar una ubicación. Si no haya ubicación, el punto por defecto seria (0, 0).

SOvalo

Crea un ovalo en la ventana, con ancho y alto.
new SOvalo(ancho, alto)
new SOvalo(x, y, ancho, alto)

SRect

Crea un rectángulo en la ventana, con ancho y alto.
new SRect(ancho, alto)
new SRect(x, y, ancho, alto)

SLabel

Crea una etiqueta en la ventana.
new SLabel(etiqueta)
new SLabel(etiqueta, x, y)

SLinea

Crea una linea en la ventana, pasando el punto inicio y el punto final.
new SLinea(x0, y0, x1, y1)

Métodos de tipo SObjeto

Estos métodos se puede usar con todos los objetos gráficos.
void moverse(dx, dy) // mueve el objeto en los axises x-y respectivamente

double darAncho()
double darAlto()

cambiarUbicacion(i, j) // cambia la ubicacion para estar en punto i-j
cambiarColor(color)

cambiarVisible(visibleSiNo)
boolean esVisible()
int darX() // devolver el parte X del punto de la esquina noroeste
int darY() // devolver el parte Y del punto de la esquina noroeste
Estos métodos se puede usar con GOval y GRect.
cambiarRelleno(rellenoSiNo)
boolean estaRelleno()
cambiarColorRelleno(color) // que color usamos para llenar
Estos métodos se puede usar con GLabel.
cambiarTexto(etiqueta)
cambiarFuente(fuente)// por ejemplo Tahoma-24
double darAscension()
double darDescenso()
Estos métodos se puede usar con GLinea.
cambiarPuntoPrincipio(x, y)
cambiarPuntoFinal(x, y)
double darPuntoPrincipioX()
double darPuntoPrincipioY()
double darPuntoFinalX()
double darPuntoFinalY()
