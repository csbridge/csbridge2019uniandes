Poner 100 (con solución)

Problema escrito por Chris

Folletos: KarelReferencia

Tu objetivo es, hacer que Karel cree una pila de 100 conos en la plaza enfrente de donde comienza. ¡¡¡Por favor!!! ¡No escribas ponerBeeper 100 veces seguidas! (psst: usar un ciclo for.)

Solución 

/**
 * Programa : Poner100
 * ------------------
 * Este programa hace que karel coloque un montón de
 * 100 conos. Buenos tiempos
 */
public class Poner100 extends EsKarel {
	
	public void run() {
		moverse();
		// Este for-loop repetirá el código dentro 100 veces.
		for(int i = 0; i < 100; i++) {
			ponerCono();
		}
		moverse();
	}
}
