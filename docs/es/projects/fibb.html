Fibbonacci (con solución)

Problema escrito por Eric Robets

En el siglo XIII, el matemático italiano Leonardo Fibonacci, como una forma de explicar el crecimiento geométrico de una población de conejos, ideó una secuencia matemática que ahora lleva su nombre. Los primeros dos términos en esta secuencia, Fib (0) y Fib (1), son 0 y 1, y cada término subsiguiente es la suma de los dos anteriores. Por lo tanto, los primeros términos en la secuencia de Fibonacci se ven así:

Fib (0) = 0
Fib (1) = 1
Fib (2) = 1 (0 + 1)
Fib (3) = 2 (1 + 1)
Fib (4) = 3 (1 + 2)
Fib (5) = 5 (2 + 3)

Escriba un programa que muestre los términos en la secuencia de Fibonacci, comenzando con Fib (0) y continúe mientras los términos sean menos de 10,000. Por lo tanto, su programa debe producir la siguiente ejecución de muestra:

Este programa debe continuar siempre que el valor del término sea menor que el valor máximo. Para hacer esto, debes usar un ciclo while que, presumiblemente con una línea de encabezado que se ve así:

while (término < VALOR_MAXIMO_PLAZO)
Tenga en cuenta que el valor máximo del término se especifica utilizando una constante nombrada.

Solución 

/ *
 * Archivo : Fibonacci.java
 * -----------------------
 * Este programa enumera los términos en la
 * secuencia de fibonacci hasta un
 * VALOR_MAXIMO_PLAZO constante, que es el término
 * más grande de Fibonacci que se incluyó en el programa.
 * /

import acm.programa. *;

clase pública Fibonacci extiende ProgramaDeConsola {


   / * Definir el término más grande que se refiere * /
	final estático privado int VALOR_MAXIMO_PLAZO = 10000;

	public void run () {
		imprimir ("Este programa enumera la secuencia de Fibonacci");
		int t1 = 0;
		int t2 = 1;
		while (t1 <= VALOR_MAXIMO_PLAZO) {
			imprimir (t1);
			int t3 = t1 + t2;
			t1 = t2;
			t2 = t3;
		}
	}

}
