Events Reference

Written by Danielle Kain, Kevin Miller

To start off, whenever you are using MouseEvents be sure to write the following in your run method:
añadirMouseListeners();
This will alert the program to react to MouseEvents. If you are using KeyListeners you'll want to include:
añadirOyentesClave();
Once you have your listeners all set up, it's time to get to the code! Below is a list of the four events that you may find useful for Breakout. For a full list, check out: this link for MouseEvents and this link for KeyListeners. 

public void ratónPresionado(EventoDelRatón e) {
	//Este método se llama cuando se hace clic con el ratón.
}

public void ratónMovido(EventoDelRatón e) {
	//Este método se llama cuando se mueve el ratón.
}
			
public void teclaPresionada(EventoClave e) {
	//Este método se llama cuando se presiona cualquier tecla.
}

esperaParaHacerClic(); //Esto hará una pausa hasta que el usuario haga clic.

Here are some examples of how to use events: 

public void ratónPresionado(EventoDelRatón e) {
	Sistema.afuera.imprimir("X: " + e.obtenerX() + "y:" + e.obtenerY());
	//Esto imprimirá las coordenadas de donde hace clic.
}

public void teclaPresionada(EventoClave e) {
	Sistema.afuera.imprimir("tecla presionada:" + e.obtenerLaClave());
	//Esto imprimirá cual tecla fue presionada
}
