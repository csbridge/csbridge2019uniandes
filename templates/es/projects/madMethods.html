Ejemplo: Método promedio, dibujar personas, mitad verde.

1. Mad Max

Completa el programa MadMax llenando el método max que devuelve el mayor de sus dos parámetros:

private int max(int i, int j)

El método de ejecución ya ha sido escrito: 

 public void run () {
    imprimir ("El máximo de 4 y 5 es:" + max (4, 5));
    imprimir ("El máximo de -4 y 4 es:" + max (-4, 4));
    imprimir ("El máximo de 1 y 2 es:" + max (1, 2));
    imprimir ("El máximo de 0 y 0 es:" + max (0, 0));
}

2. Dibujar árbol

Complete el método dibujarArbol que dibuja un árbol con la base de su tronco centrado en la posición dada (x, y):

private void dibujarArbol(int x, int y)

El método de ejecución ya ha sido escrito: 

 public void run () {
    dibujarArbol (50, 150);
    dibujarArbol (100, 300);
    dibujarArbol (500, 200);
    dibujarArbol (300, 250);
    dibujarArbol (700, 310);
}

Cada árbol debe tener un camión con un ancho de 20 píxeles y una altura de 80 píxeles. Las hojas deben tener un radio de 30 píxeles. Tu salida debería verse así:

Si terminas temprano, experimenta con cuán hermoso puedes hacer que luzca un árbol.
