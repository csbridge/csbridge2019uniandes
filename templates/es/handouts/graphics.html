Interfaces Gráficas Referencia

Escrito por Eric Roberts y Lisa Yan

Agregar y quitar del canvas

Cuando creas objetos del tipo SObjeto, necesitas agregarlos al canvas. Después de crear los objetos con sus constructores, necesitas agregarlos para que se aparezcan en el canvas. Para eso, se usa estos métodos, sin ubicación o con ubicación respectivamente.
agregar(objeto) // en la ubicación que indica el objeto
agregar(objeto, x, y) // en la ubicación indicada
También puedes quitar un objeto o todos los objetos del canvas.
quitar(objeto) // quita este objeto
quitarTodo() // quita todos los objetos que ya agregaste
Esos otros métodos de EsGraphics te puede servir también:
pausa(milisegundos) // pausa el código
SObjeto darObjetoA(x, y) // devuelve el objeto en esta ubicación

Constructores

Todos los objetos de la librería de interfaces gráficas son tipos de SObjeto, y se muestran los constructores aquí. Se puede construir sin ubicación o con ubicación respectivamente. Necesitas especificar una ubicación cuando construyes un objeto o cuando lo agregas al canvas. Si no especificas la ubicación, por defecto será (0, 0).

SOvalo

Crea un ovalo en el canvas, con ancho y alto.
SOvalo ovalo1 = new SOvalo(ancho, alto)
SOvalo ovalo2 = new SOvalo(x, y, ancho, alto)

SRect

Crea un rectángulo en el canvas, con ancho y alto.
SRect rect1 = new SRect(ancho, alto)
SRect rect2 = new SRect(x, y, ancho, alto)

SEtiqueta

Crea una etiqueta en el canvas.
SEtiqueta etiqueta1 = new SEtiqueta(etiqueta)
SEtiqueta etiqueta2 = new SEtiqueta(etiqueta, x, y)

SLinea

Crea una línea en el canvas, indicando el punto inicio y el punto final.
SLinea linea = new SLinea(x0, y0, x1, y1)

Métodos del tipo SObjeto

Estos métodos se puede usar con todos los objetos gráficos.
objecto.moverse(dx, dy) // mueve el objeto en los axises x-y respectivamente

double ancho = objecto.darAncho()
double alto = objecto.darAlto()

objecto.cambiarUbicacion(i, j) // cambia la ubicación para estar en punto (i,j)
objecto.cambiarColor(color)

objecto.cambiarVisible(visibleSiNo)
boolean visible = objecto.esVisible()
int x = objecto.darX() // devolver la parte X del punto de la esquina noroeste
int y = objecto.darY() // devolver la parte Y del punto de la esquina noroeste
Estos métodos se puede usar con SOvalo y SRect.
cambiarRelleno(rellenoSiNo)
boolean relleno = estaRelleno()
cambiarColorRelleno(color) // qué color usamos para llenar
Estos métodos se puede usar con SEtiqueta.
etiqueta.cambiarTexto(etiqueta)
etiqueta.cambiarFuente(fuente)// por ejemplo Tahoma-24
double ascenso = etiqueta.darAscenso()
double descenso = etiqueta.darDescenso()
Estos métodos se puede usar con SLinea.
linea.cambiarPuntoPrincipio(x, y)
linea.cambiarPuntoFinal(x, y)
double principioX = linea.darPuntoPrincipioX()
double principioY = linea.darPuntoPrincipioY()
double finalX = linea.darPuntoFinalX()
double finalY = linea.darPuntoFinalY()
