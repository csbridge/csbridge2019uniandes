Karel NU (con solution)

Problema escrito por Eric Roberts

Guras: Referencia de karel

Karel ha sido contratada por la ONU para venir a ayudar a construir casas. El día necesita ser salvado y Karel es claramente el mejor robot para el trabajo. Su objetivo es hacer que Karel camine por la fila inferior y construir una casa como la siguiente:

Cada vez que Karel encuentra un cono. Las siguientes imágenes muestran lo que Karel debería hacer por dos mundos de ejemplo. Puede suponer que ninguna casa se superpondrá y que todas las casas encajarán dentro del mundo. ¡No puedes asumir el tamaño del mundo o el número de casas!

Solution

/**
 * Programa: Karel NU
 * -------------------
 * Este programa hace que Karel construya casas,
 * como trabajo para la ONU.
 */
public class KarelNU extends EsKarel {
	
	public void run() {
		while(frenteDespejado()) {
			moverse();
			if(conosPresentes()) {
				construirUnaCasa();
			}
		}
	}

	/**
	 * Método: Construir Casa
	 * ----------------------
	 * Construye una sola casa centrada alrededor de la
	 * posición actual de Karel. Puede suponer que hay
	 * un cono presente en el estado de
	 * inicio. Al final del método, Karel estará mirando
	 * hacia el este en la esquina inferior derecha de
	 * la casa.
	 */
	private void construirUnaCasa() {
		recogerCono();
		mediaVuelta();
		moverse();
		girarDerecha();
		ponerTres();
		girarDerecha();
		moverse();
		girarDerecha();
		ponerTres();
		girarIzquierda();
		moverse();
		girarIzquierda();
		ponerTres();
		mediaVuelta();
		moverseALaPared();
		girarIzquierda();
	}

	/**
	 * Método: Mover a la pared
	 * ------------------------
	 * Avanza hasta que Karel golpee una pared.
	 */
	private void moverseALaPared() {
		while(frenteDespejado()) {
			moverse();
		}
	}

	/**
	 * Método: Poner tres
	 * --------------------
	 * Poner tres beepers seguidos y moverse tres veces
	 */
	private void ponerTres() {
		for(int i = 0; i < 3; i++) {
			ponerCono();
			moverse();
		}
	}

	/* el clasico */
	private void girarDerecha() {
		girarIzquierda();
		girarIzquierda();
		girarIzquierda();
	}

}
