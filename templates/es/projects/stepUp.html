Aumentar

Basado en un folleto de Eric Roberts

Folletos: Referencia de Karel

Karel comienza en el mundo a la izquierda. ¿Cómo programaría a Karel para que recoja el beeper y lo transporte hasta la parte superior de la cornisa? Karel debería dejar caer la señal acústica en la esquina de 2nd Street y 4th Avenue y luego continuar una esquina más hacia el este, terminando en 5th Avenue. Al final de tu programa, el mundo de Karel debería verse como la imagen de la derecha.


Si Karel solo conoce los comandos:

moverse()
recogerBeeper()
ponerBeeper()
girarALaIzquierda()
¿Cómo puedes hacer que Karel gire a la derecha?

Solución

Se proporciona una solución para este ejemplo, ¡para que pueda ver cómo se ve un programa completo! 

/ **
 * Programa : Step Up
 * -----------------
 * Tu primer programa de Karel ejemplo. Haz que
 * Karel rechaza el beeper que está frente a ella y
 * La coloque encima de la cornisa. Este es un
 * comentario. Tu computadora lo ignorará.
 * /
clase pública Aumentar extiende Karel {
	
	// Cuando inicies tu programa, este código será ejecutado.
	public void run () {
		moverse ();
		recogerBeeper ();
		girarALaIzquierda ();
		moverse ();
		girarALaDerecha ();
		moverse ();
		ponerBeeper ();
		moverse ();
	}

	// Definir un nuevo comando llamado girarALaDerecha.
	volante privado girarALaDerecha () {
        girarALaIzquierda ();
        girarALaIzquierda ();
        girarALaIzquierda ();
    }
}
