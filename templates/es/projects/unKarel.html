UN Karel (with solution)

Problem written by Eric Roberts

Handouts: KarelReference

Karel has been hired by the UN to come and help build houses. The day needs saving and Karel is clearly the best robot for the job. Your goal is to have Karel walk along the bottom row and build a house like the following:

Every time Karel finds a beeper. The bellow images show what Karel should do for two example worlds. You can assume that no houses will overlap and that all houses will fit inside the world. You can't assume the size of the world or the number of houses!

Solution

/**
 * Programa: ONU Karel
 * -------------------
 * Este programa hace que Karel construya casas,
 * como trabajo para la ONU.
 */
public class UNKarel extends SuperKarel {
	
	public void run() {
		while(elFrenteEsClaro()) {
			moverse();
			if(beepersPresente()) {
				construirUnaCasa();
			}
		}
	}

	/**
	 * Método: Construir Casa
	 * ----------------------
	 * Construye una sola casa centrada alrededor de la
	 * posición actual de Karel. Puede suponer que hay
	 * una señal acústica presente en el estado de
	 * inicio. Al final del método, Karel estará mirando
	 * hacia el este en la esquina inferior derecha de
	 * la casa.
	 */
	private void construirUnaCasa() {
		recogerBeeper();
		giroDeVuelta();
		moverse();
		girarALaDerecha();
		lugarTres();
		girarALaDerecha();
		moverse();
		girarALaDerecha();
		lugarTres();
		girarALaIzquierda();
		moverse();
		girarALaIzquierda();
		lugarTres();
		giroDeVuelta();
		moverALaPared();
		girarALaIzquierda();
	}

	/**
	 * Método: Mover a la pared
	 * ------------------------
	 * Avanza hasta que Karel golpee una pared.
	 */
	private void moverALaPared() {
		while(elFrenteEsClaro()) {
			moverse();
		}
	}

	/**
	 * Método: Coloque tres
	 * --------------------
	 * ¡Coloca tres beepers seguidos y mueve tres veces!
	 */
	private void lugarTres() {
		for(int i = 0; i < 3; i++) {
			ponerBeeper();
			moverse();
		}
	}

}
