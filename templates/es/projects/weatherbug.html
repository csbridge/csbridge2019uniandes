Insecto de lluvia

Creado por Nick Troccoli y Bryce Cronkite-Ratcliff

Si alguna vez usaste weather.com u otro sitio meteorológico para buscar el clima, sabes cuán útil (¡y genial!) Es poder buscar el clima en línea tanto en tu ubicación actual como en cualquier otro lugar del mundo . Y, como resultado, también es una excelente manera de aprender y practicar las devoluciones de llamada de JavaScript. En esta asignación, completará un sitio web que, utilizando JavaScript, obtiene y muestra el clima de cualquier lugar que el usuario ingrese, o la ubicación actual del usuario.

Sabemos que todos ustedes son ninjas GRect y EtiquetaG, por lo tanto, ¡hemos hecho todo el código de gráficos para usted! Lo que queremos que haga es escribir todas las devoluciones de llamada para que el sitio web funcione correctamente. Le sugerimos que divida esto en hitos y haga que cada uno de ellos funcione antes de continuar.

Hito 1: Botones

La primera tarea que debe abordar es lograr que las devoluciones de llamada de su sitio web funcionen correctamente. Su página debe tener dos botones: "Mostrar clima" y "Mostrar clima local".

Para agregar botones a la pantalla, use el método addButton () de AplicaciónDeGráficos. addButton () toma 2 parámetros: el texto que debe mostrar el botón y la función de devolución de llamada que debe ejecutarse cuando se hace clic en el botón.

Recuerde que cuando desea pasar una función como devolución de llamada, debe escribir el nombre de la función sin paréntesis.

Hito 2: El botón "Mostrar clima"

Ahora que ya tiene configurados sus botones, es el momento de abordar el primero de los dos botones: buscar el clima para el lugar que el usuario ingresa. Puede obtener el texto que el usuario escribió en el campo de texto de la propiedad de valor de su campo de texto. variable. De lo contrario, solo hay 2 funciones AplicaciónDeGráficos que necesitarás para este hito: fetchWeatherForQuery () y displayErrorMessage (). Su sitio web debe obtener el pronóstico de 7 días para el lugar que escribió el usuario. Si la búsqueda es exitosa, debe mostrar los datos meteorológicos que recupera. Tenga en cuenta que la función de devolución de llamada exitosa debe tomar un parámetro: los datos meteorológicos.

Recuerde, al obtener datos de la web, la solicitud podría fallar por varios motivos. Es por eso que debe pasar una devolución de llamada de error al obtener datos meteorológicos. Si la recuperación falla, debe mostrar un mensaje de error al usuario utilizando displayStatusMessage ().

Hito 3: El botón "Mostrar clima local"

Ahora abordemos el segundo botón: buscar el clima para la ubicación actual del usuario. En lugar de obtener el texto que ingresó el usuario, debe buscar las coordenadas de la ubicación actual del usuario (latitud y longitud). Obtener la ubicación actual requiere una devolución de llamada, ¡ya que lleva mucho tiempo localizar al usuario! Esa función de devolución de llamada debe tomar 2 parámetros: la latitud y la longitud.

Una vez que obtenga la ubicación del usuario, su sitio web debería obtener nuevamente el pronóstico de 7 días para el lugar que el usuario escribió, pero esta vez usando fetchWeatherForCoodinates (). Si la recuperación es exitosa, debe mostrar los datos meteorológicos que recupera. Tenga en cuenta que la función de devolución de llamada exitosa debe tomar un parámetro: los datos meteorológicos.

Nuevamente, recuerde que, al obtener datos de la web, la solicitud siempre puede fallar por varios motivos. Es por eso que necesita pasar una segunda devolución de llamada al obtener datos meteorológicos. Si la recuperación falla, debe mostrar un mensaje de error al usuario utilizando displayStatusMessage ().

Extensiones

Ahora que ha terminado la parte de las devoluciones de llamada, ¡intente extender los gráficos de WeatherBug! Por ejemplo, intente agregar nombres de días a la pantalla cuando se muestre el clima. Los datos meteorológicos que recuperas tienen una propiedad dayName que es el día para el que está el clima. Intente mostrar otra etiqueta para cada día que muestre el nombre del día.
