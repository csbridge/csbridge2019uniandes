Referencia de generador aleatorio

Escrito por Danielle Kain y Lisa Yan

Los generadores aleatorios le permiten agregar azar a su programa. Independientemente del tipo de objeto aleatorio que desee, primero deberá crear un generador aleatorio:
GeneradorAleatorio rgen = new GeneradorAleatorio();

Puedes hacer muchos tipos de objetos aleatorios usando RandomGenerators. Aquí hay unos ejemplos:
int aleatorioInt = rgen.siguienteInt(); //Devuelve un int aleatorio
double dobleAleatorio = rgen.siguienteDoble(); //Devuelve un doble aleatorio
Color colAlAzar = rgen.siguienteColor(); //Devuelve un color al azar
boolean aleatorioBooleano = rgen.siguienteBooleano(); //Devuelve un booleano aleatorio

Enteros

La llamada al método predeterminado (sin argumentos) devuelve un entero sin límites, que puede ser muy grande o muy pequeño. Para restringir el rango de los enteros que obtiene, puede especificar límites. Entonces, para obtener un entero entre 0 y 10, inclusive, puedes hacer lo siguiente:

int lb = 0;
int ub = 10;
int x = rgen.siguienteInt(lb, ub); // entero en el rango [0,10]

Dobles

La llamada al método por defecto devuelve un doble entre 0.0 y 1.0. Para aumentar el rango de dobles, puede especificar límites:

double y = rgen.siguienteDoble(-3, 5.5); // nota -3 se convierte de entero a doble -3.0

Booleanos

A diferencia de los enteros y los dobles, hay un solo argumento p que define el comportamiento de rgen.siguienteBooleano(p):

  • Devuelve true con probabilidad p.
  • Devuelve false con probabilidad 1-p.
  • p debe ser un doble entre 0 y 1.0.
  • El valor predeterminado de p es 0.5.
bool igualProb = rgen.siguienteBooleano();
bool probDesigual = rgen.siguienteBooleano(0.75); // devuelve verdad 3/4 del tiempo
