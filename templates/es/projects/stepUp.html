Escalar (con solucion)

Basado en un folleto de Eric Roberts

Folletos: Referencia de Karel

Karel comienza en el mundo a la izquierda. ¿Cómo programaría a Karel para que recoja el cono y lo transporte hasta la parte superior de la cornisa? Karel debería dejar caer el cono en la esquina de 2nd Street y 4th Avenue y luego continuar una esquina más hacia el este, terminando en 5th Avenue. Al final de tu programa, el mundo de Karel debería verse como la imagen de la derecha.


Si Karel solo conoce los comandos:

moverse()
recogerCono()
ponerCono()
girarIzquierda()
¿Cómo puedes hacer que Karel gire a la derecha?

Solución

Se proporciona una solución para este ejemplo, ¡para que pueda ver cómo se ve un programa completo! 

/**
 * Programa : Step Up
 * -----------------
 * Tu primer programa de Karel ejemplo. Haz que
 * Karel rechaza el cono que está frente a ella y
 * La coloque encima de la cornisa. Este es un
 * comentario. Tu computadora lo ignorará.
 */
public class Escalar extends EsKarel {
	
	// Cuando inicies tu programa, este código será ejecutado.
	public void run () {
		moverse();
		recogerCono();
		girarIzquierda();
		moverse();
		girarDerecha();
		moverse();
		ponerCono();
		moverse();
	}

	// Definir un nuevo comando llamado girarDerecha.
	private void girarDerecha () {
		girarIzquierda();
		girarIzquierda();
		girarIzquierda();
	}
}
