8 Bolas (con solución)

Problema escrito por Chris Piech.

La idea detrás de un 8-Ball es muy simple. Le haces a la bola ocho una pregunta de sí o no, y te dice la respuesta.

Excepto que la respuesta que elija se selecciona aleatoriamente de un conjunto de respuestas prefabricadas.

Una vida real 8-Ball

Escriba un programa que continuamente le pida al usuario una pregunta de sí o no, y luego selecciona aleatoriamente entre 5 respuestas enlatadas:

  • Sin duda.
  • Sí.
  • Pregunta de nuevo más tarde.
  • No.
  • Karel piensa que sí.

Aquí hay un ejemplo de ejecución del programa.

Para generar números aleatorios, cree una variable de instancia GeneradorAleatorio (una variable declarada fuera de todos los métodos) así:

private GeneradorAleatorio rgen = new GeneradorAleatorio();

A continuación, puede utilizar la variable para generar números aleatorios.

rgen.siguienteInt( max) //genera un int aleatorio en el rango [0, max)

Solución 

/ **
 * Clase : ocho bolas.
 * -----------------
 * Simula una bola ocho y respuestas sabias a las
 * preguntas de sí o no.
 * /
clase pública BolaOcho extiende ProgramaDeConsola {
	
	GeneradorAleatorio estático privado rg = new GeneradorAleatorio ();
	
	public void run () {
		while (verdadero) {
			leerLinea ("Haz una pregunta de sí o no:");
			
			int elección = rg.siguienteInt (5);
			imprimir (elección);
			si (elección == 0) {
				imprimir ("Sin duda");
			}
			si (elección == 1) {
				imprimir ("Sí");
			}
			si (elección == 2) {
				imprimir ("Pregunta de nuevo más tarde.");
			}
			si (elección == 3) {
				imprimir ("No");
			}
			si (elección == 4) {
				imprimir ("Karel cree que sí".);
			}
			imprimir ("");
		}
	}
}
