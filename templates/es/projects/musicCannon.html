Cañon de musica

Folletos: Sintaxis de JavaScript Folleto

¡Clientela! Servidores! ¡Música! ¡Tres grandes sabores que saben muy bien juntos!

Tu misión es escribir un programa cliente que presente al usuario un cuadro de texto. Cuando el usuario ingresa el nombre de una nota en el cuadro de texto y presiona la tecla 'Enter', esa nota se envía a nuestro servidor y el servidor la reproduce.

Lo primero que hay que hacer!

Navegue a su escritorio, abra la carpeta jsday3, abra la carpeta llamada musicCannon. Abre index.html y musicCannon.js en Notepad ++. También abre index.html en Google Chrome

Ahora crea un cuadro de texto

Crear un cuadro de entrada de texto en la página web del cliente. Usted agregará esto en index.html

Para crear un cuadro de entrada en HTML, use la etiqueta de entrada: a continuación se muestra un ejemplo. El atributo de marcador de posición establece el texto que aparece dentro del cuadro antes de que el usuario ingrese algo; es opcional 

<entrada tipo="texto" marcador_de_posición="¡Introduce un texto en este cuadro!">

Ahora escribe una devolución de llamada

Ahora, en el archivo musicCannon.js, escribe una función que tome un parámetro. Dentro de esa función, hagamos un empuje a nuestro servidor usando el parámetro pasado a esta función.

El código para hacer un push tiene el siguiente formato: 

$.enviar( "http: // una URL COOL", { A LLAVE: VALOR TO UTILIZAR } );

Una URL COOL, una CLAVE y un VALOR DE USO no son valores reales del programa; representan lo que tenemos que reemplazar para que este código funcione en nuestro programa específico.

Ahora, en este programa, nuestra url es 

 http://musica-cannon.meteor.com/music
, nuestra clave es la nota, y nuestro valor se pasa a la función como un parámetro.

Ahora conecte la devolución de llamada a su cuadro de texto!

Ahora que podemos enviar al servidor, debemos conectar nuestro cuadro de texto a la función que acabamos de crear para que cuando cambie la entrada del cuadro de texto, se llame a la función que escribimos para que los datos se pasen al servidor. Podemos hacer esto con un oyente de eventos, como lo hicimos con círculos al azar. Solo necesitamos agregar lo siguiente a nuestra etiqueta de entrada, donde se utiliza este valor para recuperar el texto actual dentro del cuadro de texto y pasarlo a la función 

<entrada tipo="texto" onchange="NAME_OF_FUNCTION_THAT_PUSHES (this.value)">

Asegúrese de poner el nombre correcto de la función en el código anterior

¡Ahora prueba tu código! Intente colocar una nota en minúscula que se encuentra en la escala A blues en el cuadro de texto, y escuche con atención ...
