Ondas de sonido

Inspirado por, y construido a partir de una asignación por Keith Schwarz. Creado por Chris Piech Kevin Miller y Bryce Cronkite-Ratcliff

Folletos: ArrayLists

Los seres humanos siempre están buscando nuevos sonidos, y recientemente un número creciente de artistas ha estado usando programas para crear experiencias que los instrumentos no pueden producir. Aquí hay dos ejemplos de canciones que fueron creadas con sonidos generados por computadora:

Escriba un programa que lea las notas de un archivo y reproduzca la composición para el usuario.

Audio

Hay tres métodos de audio que te permitirán generar y reproducir notas.

ListaDeArreglo<Doble> muestras= Audio.obtenerMuestrasDeNota( frecuencia, longitud)
Audio.jugar( matrizListaDeDobles)

obtenerMuestrasDeNota es un método que toma dos números, una frecuencia y una longitud en muestras (hay 44.100 muestras por segundo en sonido con calidad de CD). La frecuencia define la nota que se toca. Por ejemplo, una nota C tiene una frecuencia de 130.81 Hz y una nota G tiene una frecuencia de 196.00 Hz. Debes usar los contantes definidos. Este método devuelve un ListaDeArreglo de Dobles, que representa las muestras de la onda que hace el sonido.

Tomemos la ola de abajo como ejemplo. Tiene una frecuencia de 130.81 Hz que es una nota C. El método muestreará la onda y devolverá los dobles recogidos de la muestra, en el rango (-1, 1):


Imagen cortesía de Keith

play es un método que toma una lista de muestras (que debe ser del tipo ListaDeArreglo <Double>) y lo reproduce para el usuario.

getNotes es un método que toma el nombre de una canción y devuelve una ListaDeArreglo <String> con todas las notas de la canción. Las letras en ListaDeArreglo pueden ser notas en mayúsculas o minúsculas (ag y p para pausa). Aquí hay un ejemplo ListaDeArreglo: {a, a, G, f, p, B}.

1) tocar una nota

Para comenzar, intente generar una única nota en C de longitud LONGITUD_DE_LA_NOTA y reprodúzcala al usuario. Luego trata de tocar algunas notas diferentes. Intente tocar las notas E, C, C. Finalmente, toque una nota una octava hacia arriba y asegúrese de que pueda notar la diferencia.

2) Haz un programa musical

Este proyecto está abierto. En este punto puedes programar lo que quieras. ¿Te gustaría crear un piano gráfico? ¿Qué tal generar sonidos más complejos? Vea las extensiones e intente e implemente todas las que desee.

Consejos y trucos

Jugando una octava
Para jugar una octava, todo lo que tienes que hacer es duplicar la frecuencia. Por ejemplo, si el C original es 130.81, entonces, una octava más alta en C es 2 * 130.81 = 261.62

Métodos de uso
Puede que le resulte más fácil escribir un método para tocar cada nota:

private void jugarG() {
   ListaDeArreglo<Doble> ejemplo = Audio.obtenerMuestrasDeNota(G, LONGITUD_DE_LA_NOTA);
   Audio.jugar(ejemplo);
}

Comparar cuerdas

Para comparar cadenas, no use el operador ==. En su lugar, debe utilizar:

s1.es_igual_a( s2)

¡Eso es todo! Bienvenido al mundo de ArrayLists y música de computadora. Si terminas temprano intenta una extensión.

Extensión 1: Escribe tu propia música

Intenta componer una melodía a partir de diferentes notas. Añadir frases de nivel superior a las canciones. Usa bucles y métodos para hacer sonidos complejos.

Extensión 2: Acordes

Dos tocan dos (o más) notas simultáneamente, simplemente sumas cada valor en la muestra. Luego, para hacer que el volumen sea normal, divida la muestra resultante por su valor máximo.

Puede hacer sonidos más ricos, más cercanos a los utilizados por Tycho y Pretty Lights (en los ejemplos al principio).

Extensión 3: Interfaz gráfica

Haga una interfaz gráfica utilizando una cuadrícula de GRectos donde cada fila representa una nota (C, D, G, E, A) y cada columna representa un punto en el tiempo. Si un GRect está activado (es de color azul), debe tocar esa nota en ese momento.
