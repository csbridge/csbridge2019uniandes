Mouse Listeners

Cuando usas eventos de mouse, es necesario poner este método en su primera linea del método run(). Esto alertará al programa para que reaccione a los eventos de MouseEvento.
agregarMouseListeners();
Aquí tenemos todos los métodos que escribirás para usar el movimiento y uso del mouse en su código. Recuerda que no se debe llamar esos métodos. Cuando lo escriben, los métodos van a llamarse cuando el evento pase.
public void mouseMovido(MouseEvento e) // cuando muevas mouse
public void mouseArrastrado(MouseEvento e) // cuando arrastres mouse
public void mousePulsado(MouseEvento e) // cuando pulses en mouse
public void mouseSoltado(MouseEvento e) // despues de pulsar cuando el mouse solte
public void mouseClickeado(MouseEvento e) // click (pulsado y soltado en misma ubicación)
public void mouseSale(MouseEvento e) // cuando mouse sale del canvas
public void mouseEntra(MouseEvento e) // cuando mouse entra el canvas
Esto es un ejemplo de escribir uno de esos métodos. 
public void mousePulsado(MouseEvento e) {
	imprimir("X: " + e.darX() + "y:" + e.darY());
	//Esto imprimirá las coordenadas de donde presionas mouse.
}
