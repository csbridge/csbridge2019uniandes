Referencia del programa de consola

Los programas de consola permiten la interacción entre su programa y el usuario. Para escribir un programa, asegúrese de que su programa extienda ProgramaDeConsola:

public class Ejemplo extends EsConsole {

Imprimir

imprimir(mensaje); //Imprime un mensaje a la consola.

Leer

Una cosa interesante que los Programas de la Consola te permiten hacer es obtener información del usuario. A continuación se presentan algunos ejemplos:
leerLinea(aviso); //Lee texto de usuarios y devuelve
leerInt(aviso); //Lee un entero y lo devuelve.
leerDouble(aviso); //Lee un double del usuario y lo devuelve.

Ejemplo

Aquí hay un ejemplo que lee un número entero y lo imprime dividido por dos
public class Ejemplo extends EsConsole {
   public void run() {
      int numeroDeUsuario = leerInt("por favor digita un numero");
      imprimir(numeroDeUsuario / 2);
   }
}
