Constantes

Puede observar que el código de inicio para este proyecto tiene las líneas:

private static final int TAMAÑO = 100;
private static final int BRECHA = 10;
que se escriben fuera del método de ejecución (como el generador de números aleatorios). Estas líneas definen constantes: variables cuyos valores nunca cambian y como se definen fuera de un método, puede leer su valor desde cualquier punto del programa. Aunque se podría escribir esta ilusión óptica utilizando los números 100 y 10 directamente, usando las constantes hace que su programa mucho más fácil de leer.

Doble ciclo for

Este problema requiere usar un ciclo for dentro de un ciclo for. Recuerde utilizar una variable de iterador diferente para el bucle interno. Por ejemplo, si su bucle externo es de la forma

for(int i = 0; i < 100; i++)
El bucle en el cuerpo debe iterar a través de una variable distinta de i.

Extensión

Si tienes tiempo, ¡intenta crear esta ilusión óptica! ¡Tenga en cuenta que todos los cuadrados son del mismo tamaño y todas las líneas son horizontales! Piensa en cómo descomponer tu solución.

Si está atascado, intente dibujar cada fila con el primer cuadrado negro comenzando siempre en x = 0. Luego, intente averiguar cómo crear el patrón de desplazamiento.
