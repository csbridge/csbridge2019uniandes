Primera parte: Pausa del alfabeto inglés

En HagaClicEnNPausa.java, escriba un programa de gráficos que HagaClicEnNPausa.java el alfabeto inglés (de la A a la Z). También agregue funcionalidad para que cuando el usuario haga clic en algún lugar de la pantalla, aparezca una etiqueta de "pausa ..." donde el usuario hizo clic.

Reglas generales:

  • Utilice un retraso de 100 milisegundos o menos entre cada carácter alfabético.
  • Agregue una pausa de 1 segundo después de cada clic del usuario.
  • Comience con números animados antes del alfabeto para establecer su ciclo principal de animación.
  • Agregue una etiqueta haciendo_una_pausa... que aparezca donde el usuario hizo clic.

Animando a los personajes.

Ya que estamos animando letras y no enteros, deberíamos hacer un seguimiento de una variable de clase char. Un char es un solo carácter largo (por lo tanto, es diferente de String, que puede ser de múltiples caracteres). Para establecer un texto de GLabel a un solo carácter:

char c = 'do';
etiqueta.colocaUnaEtiqueta(c + "");

La ventaja de usar un char sobre una cadena es que cada carácter tiene una representación de representación numérica única, utilizando algo que llamamos ASCII. Así que si convertimos el carácter A en un entero:

Sistema.afuera.imprimir((int) 'UNA'); // devuelve 65

Asimismo, la representación numérica de Z es 65 + 26 - 1 = 90. La asignación de char a int se decide por los códigos ASCII.

Utilice esta información para idear un método para recorrer en bucle las letras de la A a la Z, luego vuelva a A.

Banderas de eventos

Agregue un dispositivo de escucha del mouse para que cuando un usuario haga clic, el contador deje de aumentar temporalmente, y el signo de "pausa ..." aparezca en la ubicación en la que el usuario hizo clic. Después de un segundo, el signo de "pausa ..." debería desaparecer y el contador debería continuar aumentando.


¡¡¡Será difícil implementar todo esto en ratónPresionado()!!! No puede pausar dentro de un método de escucha como ratónPresionado() o ratónMovido(), porque estos métodos están diseñados para actuar rápidamente. En su lugar, es necesario utilizar indicadores de eventos.

Una bandera es una variable booleana que el programador establece en true cuando ocurre un evento, y se establece en false cuando el evento finaliza. En la práctica, un programador establece los indicadores de evento en verdadero en los escuchas del mouse, verifica si son verdaderos en el ciclo de animación principal y luego realiza alguna acción después de ver que el indicador está activado. Aquí hay un ejemplo de un bucle de animación principal que también verifica una bandera: 

 privado boolean banderaDeEvento = false;
public void run () {
    while (verdadero) {
        if (banderaDeEvento) {
            hacerAlgoEspecial ();
            banderaDeEvento = falso; // restablecer la bandera una vez que haya terminado
        }
        hacerAlgoNormal ();
        pausa (RETRAER);
    }
}

Vacío público ratónMovido (EventoDelRatón e) {
    banderaDeEvento = true; // encender la bandera
}

Con este nuevo conocimiento, termina la primera parte.

Parte dos: El increíble salto de Karel

En HagaClicEnNJump.java, implemente un programa donde Karel (abajo) se mueva de lado a lado para siempre. Cuando el usuario hace clic, Karel salta alto, aterriza y continúa moviéndose de lado a lado.

Reglas generales:

  • Guarde el png de Karel en la carpeta src donde se encuentran sus archivos java. Puede encontrar la ubicación de su proyecto en Eclipse, haciendo clic derecho en la carpeta del proyecto y seleccionando "Propiedades".
  • Actualice la carpeta en Eclipse hasta que vea karel.png visible en el Explorador de proyectos.
  • Karel debería moverse con suavidad, así que una vez cada 50 milisegundos (o incluso con más frecuencia).

Pautas para saltar:

  • El salto total debe durar dos segundos.
  • Mientras Karel salta, no debería poder reiniciar el salto.
  • Comienza haciendo que Karel salte verticalmente hacia arriba y hacia abajo, a velocidad constante.
  • Entonces, considere agregar en gravedad. ¿Cuál debería ser tu velocidad de arranque?
  • Finalmente, agrega movimiento horizontal mientras Karel está saltando.
