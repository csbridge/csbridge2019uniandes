Folletos: Sintaxis de JavaScript Folleto

Me gustan los círculos al azar, así que vamos a hacer algunos!

El programa comienza con una página web. Cada vez que un usuario haga clic en esa página web, aparecerá un nuevo círculo aleatorio.

Tu tarea es completar la función añadirCírculoAleatorio en randomCircles.js y ejecutar en main.js. Abra ambos archivos en "Notepad ++"; se pueden encontrar dentro de la carpeta "js" dentro de "randomCircles" dentro de "jsday1" en su escritorio. Esta función dibujará un círculo de radio aleatorio y color aleatorio en una posición aleatoria en el lienzo. ¡Tenga cuidado de restringir la posición y el tamaño del círculo para que siempre esté completamente en la ventana!

Día 1 Hito

Cuando se abra la página, haga que muestre 20 círculos al azar.

Día 2 Hito

Haga que aparezca un círculo aleatorio cuando el usuario (usted) haga clic en la página.

Nota: para dibujar círculos en la pantalla, necesitará las funciones en la referencia GOval al final de este documento. ¡Échales un vistazo!

Extensión 1

Haga que el círculo aleatorio se centre en el lugar donde el usuario hizo clic.

Extensión 2: Extreme Atrápame si puedes

Haz que la bola se mueva a una nueva ubicación aleatoria cuando el ratón la toca

Depuración

Si algo no parece funcionar bien, recuerde que la Consola de Javascript a veces puede tener mensajes de error útiles que explican cómo corregir su código. ¡Haz un hábito de abrirlo!

Referencia GOval

Aquí hay algunas funciones útiles que puede usar para ayudar a crear sus círculos al azar.

Hacer un nuevo círculo 

var miOvalFresco = new GOval();

Establecer propiedades de ese círculo: 

// Establecer el color de relleno del óvalo para colorear miOvalFresco.establecerColor(color); // Establecer el ancho y la altura del óvalo en píxeles miOvalFresco.establecerTamaño(ancho, altura); // Establezca la posición de la esquina superior izquierda del cuadro delimitador de este óvalo // ax, y, donde xey son coordenadas de píxel. miOvalFresco.escogerLocalización(x,y);

Añade tu círculo al lienzo: 

// obtener ancho de lienzo lona.anchoDeDesplazamiento; // devuelve 1000 (por ejemplo) // obtener la altura del lienzo lona.alturaDeDesplazamiento; // Devuelve 800 (por ejemplo) // añadir miOvalFresco al lienzo lona.añadir(miOvalFresco);
