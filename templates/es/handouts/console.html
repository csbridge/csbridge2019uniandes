Console Program Reference

Written by Danielle Kain, Kevin Miller

Console Programs allow for interaction between your program and the user. In order to write a program, make sure your program extends ProgramaDeConsola: 
public class Ejemplo extends ProgramaDeConsola {

One cool thing that Console Programs let you do is get input from the user. Below are some examples: 
leerLinea(Cuerda mensajeRápido); //Lee una cadena de usuarios y devuelve
leerInt(Cuerda mensajeRápido); //Lee en el usuario y lo devuelve.
leerDoble(Cuerda mensajeRápido); //Lee un doble del usuario y lo devuelve.
imprimir(Cuerda imprimirMensaje); //Imprime un mensaje a la consola.
