Ir al centro

Problema escrito por Chris y Julia.

En este programa movemos un cuadrado desde la izquierda al centro de la pantalla.

Solución

public class IrAlCentro extends EsGraphics {

    private static final int TAMANO = 20;
    private static final int RETRASAR = 100;
	
	public void run() {
		// Dibuja un cuadrado a la izquierda de la pantalla.
		GRect rect = new GRect(TAMANO, TAMANO);
		double x = 0;
		double y = (darAlto() - rect.darAlto()) / 2;
		rect.cambiarRelleno(true);
		agregar(rect, x, y);
		
		// Mover hasta el centro
		// rect.darX () devuelve la coordenada x del rectángulo
		double maxX = (darAncho() - rect.darAncho()) /2;
		while(rect.darX() < maxX) {
		    // moverse un pixel a la derecha
			rect.moverse(1, 0);
			// pausa de animación
			pausa(RETRASAR);
		}
	}
}
