Números ordenados

Escrito por Chris Piech

Folletos: ArrayLists
Ejemplos: cohetes

Pida al usuario que tenga menos de 10 números, luego imprima esos números en orden ordenado.

Almacena los números que el usuario ingresó en una ListaDeArreglo. Puedes ordenar el contenido de un ListaDeArreglo usando: 

 Colecciones.ordenar ( listaDeArreglo);

Consejos

Para crear una lista matricial de números use: 

 ListaDeArreglo <Entero> miLista = nuevo ListaDeArreglo <Entero> ();


Recuerde que puede leer un intenger del usuario usando: 

 int número = leerInt ( rápido);


Para agregar un elemento a una lista: 

 miLista.añadir ( nuevoValor);


Para obtener un elemento de una lista: 

 miLista.obtener ( índice);


Para recorrer todos los elementos de una lista: 

 para (int i = 0; i <miLista.tamaño (); i ++) {
    miLista.obtener (i);
}
