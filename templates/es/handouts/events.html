Events Reference

Written by Danielle Kain, Kevin Miller

Para comenzar, cuando esté utilizando MouseEvents, asegúrese de escribir lo siguiente en su método de ejecución:
agregarMouseListeners();
Esto alertará al programa para que reaccione a los MouseEventos. Una vez que haya configurado a todos sus oyentes, ¡es hora de llegar al código! A continuación se muestra una lista de los cuatro eventos que pueden ser útiles para Breakout. Para una lista completa, echa un vistazo a: 

public void mousePulsado(EventoDelRatón e) {
   //Este método se llama cuando se hace clic con el mouse.
}

public void mouseMovido(EventoDelRatón e) {
   //Este método se llama cuando se mueve el mouse.
}

esperarClic(); //Esto hará una pausa hasta que el usuario haga clic.

Here are some examples of how to use events: 

public void mousePulsado(EventoDelRatón e) {
   imprimir("X: " + e.darX() + "y:" + e.darY());
   //Esto imprimirá las coordenadas de donde hace clic.
}
