Tu trabajo

Se le proporciona un método llamado cargaSecreta() que devuelve un mensaje secreto como ListaDeArreglo <String>. Cada cadena en el ListaDeArreglo es una palabra en el mensaje.

Imprima cada una de las picaduras en la lista de arreglos (en orden) para ver el mensaje secreto.

revisión

Para iterar sobre un ListaDeArreglo, use un ciclo for. Por ejemplo, este código calcula la suma de todos los valores en una ListaDeArreglo de Dobles.

ListaDeArreglo<Doble> valores = obtenerValores();
int suma = 0;
for(int i = 0; i < valores.tamaño(); i++) {
   suma = suma + valores.obtener(i);
}
imprimir(suma);
