Diversión en bucle

Escrito por Lisa Yan

Estos son programas de práctica para ayudarlo a familiarizarse con los bucles en Java. Se ordenan de lo más fácil a lo más difícil.

Actualización de bucle

Recordemos la estructura general de un ciclo for: 

para (int i = 0; i <15; i ++) {
    // algun codigo ...
}

El bucle anterior se ejecuta durante 15 iteraciones. Puedes analizar el significado de un bucle desde la primera línea:

  • Crea un contador i que comienza en 0.
  • En cada iteración del bucle, verifique si i < 15.
  • Si i < 15, ejecute el código dentro del bucle. Mientras ejecuta el código de bucle, puede utilizar el valor de i. Así, por ejemplo, la primera vez que se ejecuta el bucle, el valor de i es cero.
  • Al final de la ejecución del código de bucle, aumente i++.
  • Si !(i < 15), salga del bucle. Por lo tanto, la última vez que se ejecuta el código de bucle, el valor de i es 14, e i se incrementa a 15. Luego falla la verificación de i < 15 y el código de bucle no se ejecuta.

¡Precaución!¡Es muy difícil detectar errores de programación en la condición de bucle! Por lo tanto, siempre sugerimos que diseñe el bucle más básico e incremente el contador del bucle en uno (es decir, i++). ¡Y luego no cambie el valor del contador de bucles dentro del propio bucle!

  • El siguiente código solo se ejecutará cuatro veces !!! ¿Ves por qué? 
    • // ¡No hagas esto! ¡Esto es difícil de depurar!
para (int i = 0; i <10; i ++) {
    i + = 2; // ¡¡¡malo !!!
    imprimir (i);
}
  • Una mejor manera de hacer el código anterior: 
    • // Utiliza otra variable para mantener seguro el contador de bucles.
para (int i = 0; i <10; i ++) {
    int j = i + 2; // no cambia el valor de i
    imprimir (j);
}

Los siguientes ejercicios tratan sobre el uso del contador de bucle for i.

Diversión en bucle 1

Trate de producir la siguiente salida con un bucle. Imprime los primeros diez números impares, comenzando desde 1.

Extensión: ahora intente imprimir los primeros diez números pares, comenzando desde 0.

Diversión en bucle 2

Trate de producir la siguiente salida con un bucle. Imprima los primeros diez números pares en orden decreciente.

¿Puede resolver este problema sin cambiar la condición de bucle del problema anterior? Es decir, comience desde el siguiente ciclo for el código: 

para (int i = 0; i <10; i ++) {
    // algun codigo ...
}

¿Cómo resolvería el problema utilizando la siguiente condición de bucle? 

para (int i = 0; i <20; i ++) {
    // algun codigo ...
}

Diversión en bucle 3

Use un bucle, pero imprima todo en una sola línea. Recuerde que imprimir() agrega un salto de línea / nueva línea, mientras que impresión() no lo hace.

Imprime los cuadrados, comenzando desde 1.

¡Intenta evitar una coma al final de la última casilla!

Diversión en bucle 4

Tiempo para ciclos for! Imprime las coordenadas de una cuadrícula de 5x5.

Sugerencia: pruebe la siguiente estructura de código: 

para (int i = 0; i <5; i ++) {
    para (int j = 0; j <5; j ++) {
        // algun codigo ...
    }
}

Práctica adicional: Ciclos while

¿Quieres más práctica con ciclos while? Haga todos los ejercicios anteriores con un ciclo while que en lugar de un ciclo for. ¡Buena suerte!
