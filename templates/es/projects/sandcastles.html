Castillos de arena

Folletos: Consola, Generador Aleatorio
Ejemplos trabajados: E = MC2, Fibbonacci, Carbono

Escriba los siguientes tres programas para calentarse a las variables Java.

1. hola mundo

Escriba un programa de consola que muestre hello world (o cualquier mensaje que desee):

2. Peso de la luna

El usuario (un terrícola) ingresa su peso en la tierra. Su programa imprimirá el peso del usuario en la luna, que es el 16.5% de su peso en la tierra.

3. números aleatorios

Escribe un programa que imprima 1,000 números al azar entre 0 y 100.

El archivo NúmerosAlAzar.java tiene un código que genera un solo número aleatorio. 

 clase pública NúmerosAlAzar extiende ProgramaDeConsola {
	
   // Un generador de números aleatorios.
   privado GeneradorAleatorio rgen = new GeneradorAleatorio ();

   public void run () {
      // cambia este código para imprimir 1000 números al azar
      // en el rango de 0 a 100.
      int ejemplo = rgen.siguienteInt (0, 10);
      imprimir (ejemplo);	
   }
}

Para generar números aleatorios, cree una variable de instancia GeneradorAleatorio (una variable declarada fuera de todos los métodos) así:

private GeneradorAleatorio rgen = new GeneradorAleatorio();

A continuación, puede utilizar la variable para generar números aleatorios.

rgen.siguienteInt( min, max) //genera un int aleatorio en el rango (min, max) inclusive
