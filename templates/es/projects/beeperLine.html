Fila de Conos

Escrito por Chris Piech

Tu objetivo es llenar la primera fila del mundo de Karel con beepers. Puede suponer que Karel comienza en la esquina inferior izquierda, mirando hacia el este. Por ejemplo, si ejecuta su programa en el mundo de la izquierda, debería producir el mundo de la derecha.

Tu programa también debería funcionar para mundos de diferentes tamaños. Por ejemplo, si ejecutamos el mismo programa EXACTO en este mundo un poco más grande, Karel debería poder colocar una línea completa de avisadores sin fallar. Esto significa que no podemos usar un ciclo for ¡No sabemos de antemano cuán grande será el mundo!

Solución 

/**
 * Programa : FilaDeConos.
 * -----------------------
 * Llene la primera fila con conos. Supone que
 * Karel comienza en la esquina inferior izquierda.
 * De la pantalla, mirando hacia el este.
 * /
public class FilaDeConos extends EsKarel {
	
	public void run () {
		while (frenteDespejado()) {
			ponerCono();
			moverse();
		}
		// Esta línea es necesaria para colocar el cono final. El numero
		// Las veces que Karel se mueve es menos que la cantidad de veces Karel
		// pone un beeper (si el mundo tiene cinco cuadrados de ancho, colocamos
		// 5 cono pero solo se mueven 4 veces)
		ponerCono();
	}
}
