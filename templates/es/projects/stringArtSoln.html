Arte de cuerdas

Basado en una demostración de Eric Roberts.

Al dibujar repetidamente líneas, podemos hacer una forma de aspecto curvo. Considere el siguiente ejemplo donde imaginamos que los puntos se espacian uniformemente a lo largo de la parte inferior e izquierda de un lienzo. Si conectamos el punto superior en el borde izquierdo al punto más a la izquierda en el borde derecho, seríamos una línea como la imagen del extremo izquierdo. Si luego conectamos los próximos 10 puntos, obtendremos una imagen en el medio y si conectamos todos los puntos, obtendremos la imagen de la derecha.

Escriba un programa que cree la "curva" en la imagen de arriba a la derecha usando líneas rectas.

Solución 

 Clase pública ArteDeCuerdas extiende ProgramaDeGraficos {

	/ ** El tamaño de la pantalla ** /
	public static final int ANCHO_DE_APLICACIÓN = 400;
	public static final int ALTURA_DE_APLICACION = 400;
	
	/ ** El número de píxeles entre los puntos finales de la línea ** /
	public static final int ESPACIADO_ENTRE_LÍNEAS = 20;
	
	/ ** El número de líneas que tendremos que dibujar ** /
	public static final int NUM_LINEAS = ANCHO_DE_APLICACIÓN / ESPACIADO_ENTRE_LÍNEAS;

	public void run () {
		para (int i = 0; i <NUM_LINEAS; i ++) {
			int compensar = i * ESPACIADO_ENTRE_LÍNEAS;
			int comienzoX = compensar;
			int empezarY = obtenerAltura ();
			int finalX = 0;
			int finalY = compensar;
			LíneaG línea = nueva LíneaG (comienzoX, empezarY, finalX, finalY);
			añadir (línea);
		}
	}
}
