Mitad verde

Escrito por Chris Piech

Folletos: Referencia Gráfica

Vamos a crear 1000 círculos en posiciones aleatorias en la pantalla. Los círculos más cercanos a la esquina inferior izquierda serán azules y los círculos más cercanos a la esquina superior derecha serán verdes. Este es un ejemplo de ejecución del programa.

Cada círculo es de 20 por 20 píxeles.

Solución 

 public class CirculoAlAzar2 extends EsGraphics {
	public static final int APPLICATION_WIDTH = 500;
	public static final int APPLICATION_HEIGHT = 500;
	public static final int TAMANO = 20;

	public void run () {
		for (int i = 0; i <1000; i ++) {
			int x = intAleatorio(darAncho()-TAMANO);
			int y = intAleatorio(darAlto()-TAMANO);

			GOval o = new GOval (x, y, TAMANO, TAMANO);
			o.cambiarRelleno (true);
			o.cambiarColor (darColor (x, y));
			agregar (o);
			pausa (2);
		}
	}

	private Color darColor (int x, int y) {
		if (esVerde (x, y)) {
			return Color.GREEN;
		}
		return Color.BLUE;
	}
	
	private boolean esVerde (int x, int y) {
		return x> y;
	}
}
