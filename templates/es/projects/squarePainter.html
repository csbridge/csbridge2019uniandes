Karel quiere hacer algo de arte real. Ayúdala a llenar cualquier mundo rectangular con cuadrados alternados de azul y verde.


Nota: Es posible que tenga que usar alguna lógica adicional para este problema. Aprenderás más sobre esta lógica mañana.

Para verificar lo contrario de cualquier condicional, use el "bang", comúnmente conocido como el signo de exclamación: 

// comprueba que la esquina en la que te encuentras NO es de un color determinado
if (! colorDeLaEsquinaEs (color)) {
    ...
}
