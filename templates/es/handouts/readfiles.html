Como Leer Archivos

Los programas de EsGraphics y EsConsole tiene dos métodos extras para leer archivos. Esos dos métodos devuelven lista de lineas y de palabras respectivamente.
ArrayList<String> lineas = leerLineasDelArchivo(nombreDeArchivo)
ArrayList<String> palabras = leerPalabrasDelArchivo(nombreDeArchivo)
Esto es un ejemplo de imprimir las lineas en un archivo. 
public void imprimirArchivo() {
	ArrayList lineas = leerLineasDelArchivo("res/nombres.txt");
	for (int i = 0; i < lineas.size(); i++) {
		imprimir(lineas.get(i));
	}
}
