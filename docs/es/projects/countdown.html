cuenta regresiva

Escrito por Lisa Yan

Folletos: Referencia Gráfica

Este es un programa extra! Está pensado para ser un poco desafiante, pero completar este programa definitivamente te ayudará a escribir tu proyecto final.

Escriba un ProgramaDeGraficos que muestre un contador de tiempo desde diez, una vez por segundo, y muestre un mensaje una vez que el contador llegue a cero.

Configurar el temporizador

Para implementar este temporizador, necesitaremos una representación lógica y una representación visual de nuestro valor de cuenta regresiva, que se proporcionan mediante dos variables de clase:

private EtiquetaG etiquetaDeCuentaAtrás; // visualización del valor del temporizador
private int cuenta_regresiva; // valor del temporizador real

Elija una fuente grande para su temporizador y comience mostrando la hora de inicio (TOTAL_TIME) en la pantalla. Puede convertir enteros en cadenas agregando el siguiente método a su código: 

 Cuerda pública formatoIntACadena (int x) {
    devuelve "" + x;
}

Implementar la cuenta atrás.

Escriba su bucle de animación que actualice las representaciones de la cuenta regresiva lógica y visual cada segundo.
¡No agregue nuevos GLabels! Solo sigue cambiando el texto del anterior usando el método de EtiquetaG colocaUnaEtiqueta().

// Establece el texto de etiquetaDeCuentaAtrás como "hola mundo"
etiquetaDeCuentaAtrás.colocaUnaEtiqueta("Hola Mundo");

Recuerde que el retraso se cuenta en milisegundos y queremos que el temporizador funcione por segundo.

Formato de su temporizador

Zero-padding

Ya que este es un temporizador, queremos rellenar con cero nuestros números, lo que significa agregar un cero al frente de un número para que permanezca con el mismo ancho. ¿Cómo actualizaría la representación visual de su temporizador de cuenta regresiva para mostrar lo siguiente?

Sugerencia: formatoIntACadena() los cambios en formatoIntACadena().

Fuentes de ancho fijo

Dependiendo de la elección de la fuente, es posible que su temporizador aún no se alinee, incluso después del cero relleno. ¡Esto es algo con lo que luchan muchos programadores orientados a los gráficos! Asegúrese de elegir una fuente de ancho fijo, donde cada carácter / dígito tenga el mismo ancho. Ejemplos comunes son Consolas, Mónaco y Courier.

Terminando

Una vez que el temporizador llega a cero, muestra un mensaje. ¡Puede ser cualquier cosa!

Bonus: Añadir minutos

Ahora agregue minutos a su pantalla de cuenta regresiva, de modo que el temporizador muestre 1:00, 0:59, 0:58, etc.

Reglas generales:

  • Intente mantener una sola variable que registre su tiempo restante en segundos. Evita el seguimiento de minutos y segundos por separado.
  • Separe su única variable en minutos y segundos cuando muestre el contador. Considera definir métodos para que tu lógica sea muy clara.
  • Para depurar rápidamente, acelere su temporizador reduciendo el tiempo de pausa.
  • ¡Recuerda la función "mod"! Ejemplo: 5 % 2 = 1

Bonus de bonificación: ¿Cómo agregarías milisegundos? La convención de visualización es: 1: 03.234.
