Números ordenados

Escrito por Chris Piech

Folletos: ArrayLists
Ejemplos: cohetes

Pida al usuario que tenga menos de 10 números, luego imprima esos números en orden ordenado.

Almacena los números que el usuario ingresó en una ArrayList. Puedes ordenar el contenido de un ArrayList usando: 

 Collections.sort(miLista);

Consejos

Para crear una lista matricial de números use: 

 ArrayList<Integer> miLista = new ArrayList<Integer>();


Recuerde que puede leer un entero del usuario usando: 

 int numero = leerInt(mensaje);


Para agregar un elemento a una lista: 

 miLista.add(nuevoValor);


Para obtener un elemento de una lista: 

 miLista.get(indice);


Para recorrer todos los elementos de una lista: 

 for (int i = 0; i <miLista.size(); i ++) {
    int temp = miLista.get(i);
    imprimir(temp);
}
