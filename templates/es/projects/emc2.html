E = MC2 (con solución)

Problema escrito por Chris, descubierto por Einstein

Uno de los descubrimientos humanos emocionantes es que la masa y la energía son intercambiables y están relacionadas por la ecuación E = M × C 2. Albert Einstein descubrió esta relación hace casi 100 años y ahora sabemos que el sol produce su calor al transformar pequeñas cantidades de masa en grandes cantidades de energía.

Escriba un programa que continuamente lea en masa del usuario y luego produzca la cantidad de energía. Muestra tu trabajo de la siguiente manera:

Si le pide al usuario que ingrese kilogramos y utiliza una velocidad de luz igual a 299792458 (que está en metros por segundo), el resultado del cálculo de E = M × C 2 estará en Jouls.

Tenga en cuenta que 3.0E8 es una notación científica para 3.0 × 10 8... que es 300,000,000. No necesita hacer nada para que Java imprima dobles en notación científica. Lo hará automáticamente cuando los números sean lo suficientemente grandes.

Dado que la velocidad de la luz nunca cambiará, tiene sentido declararla como una constante.

Solución 

/ **
 * Programa : EMC2
 * --------------
 * Este programa ayuda a los usuarios a calcular la
 * cantidad de energía que podrían obtener si
 * Transformar su masa. Gracias Einstein!
 * /
clase pública EMC2 extiende ProgramaDeConsola {
	
	// Esto declara una constante ... nunca puede cambiar valor.
	// Velocidad de la luz en m / s
	final estático privado doble C = 299792458;
	
	public void run () {
		while (verdadero) {
			// Lea la masa en el usuario.
			doble masaEnKg = leerDoble ("Introducir kilos de masa:");
			
			// Calcula energia
			doble energíaEnJulios = masaEnKg * C * C;
			
			// Mostrar el trabajo al usuario.
			imprimir ("e = m * C ^ 2 ...");
			imprimir ("m =" + masaEnKg + "kg");
			imprimir ("C =" + C + "Sra");
			imprimir (energíaEnJulios + "julios de energía!");
			imprimir ("");
		}
	}
}
