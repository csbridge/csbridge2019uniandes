JawaScript: ¡Empezando la fiesta de JavaScript en el desierto!

Escrito por Bryce Cronkite-Ratcliff

Folletos: Sintaxis de JavaScript Folleto

¡La mejor manera de aprender un nuevo idioma es tratar de decir algo en él! Abra el programa "Notepad ++" y abra el archivo "problems.js" dentro de la carpeta jawascript dentro de la carpeta jsday1 en su escritorio. En este archivo, hay algunas funciones de JavaScript que deben implementarse. Escriba su código JavaScript en cada función de acuerdo con las instrucciones sobre cada función.

Para ejecutar su código, navegue hasta el archivo "index.html" dentro de la carpeta jawascript y ábralo en google chrome. Presione Ctrl-Shift-J para abrir la consola del desarrollador donde estará visible la salida de su programa. También puede acceder a la consola desde la interfaz de Chrome como se muestra aquí. Importante: siempre que cambie su código, asegúrese de GUARDAR el archivo que modificó y, a continuación, REEMPLAZAR la página del proyecto en chrome.

Asegúrese de revisar el folleto de Sintaxis de JavaScript (que se encuentra en la parte superior de esta página) para asegurarse de que Javascript tenga el formato correcto.

¡Buena suerte!
