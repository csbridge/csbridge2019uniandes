Encontrar pi

Problema escrito por Eric Roberts.

Pi (π) es una cosa matemática de verdadera belleza. Aparece en muchas ecuaciones, sobre todo en la ecuación para el área de un círculo, el pdf de una distribución normal y, finalmente, y oh, tan maravillosamente, en la Identidad de Euler.

Pero, ¿cómo sabemos que el valor de π es 3.14159265359 ...? Hay varias formas de calcular pi. En este problema mostramos cómo hacerlo con dardos.

El nombre de este niño también es Pi. Este problema no está relacionado (aparte del nombre) con él.

Imagínate arrojando dardos al círculo amarillo de abajo a la izquierda. Si lanzas al azar una serie de dardos en el tablero de dardos, algunos caerán dentro del círculo amarillo y otros en el área gris que está afuera. Si cuenta el número de dardos que caen dentro del círculo y el número que cae dentro del cuadrado, la proporción de esos números debe ser proporcional al área relativa de las dos figuras.

Debido a que el área del círculo es π y la del cuadrado es 4, la fracción que cae dentro del círculo debe aproximarse a π / 4.

Solución 

/**
 * Encontrar pi
 * ------------
 * Encuentra el valor de pi lanzando dardos
 * aleatorios a un círculo unitario inscrito en un
 * cuadrado. La fracción que aterriza en el círculo.
 * Debe ser igual a pi / 4.
 * /
La clase pública LocalPi extiende el ProgramaDeConsola {
	
	public void run () {
		int numEnCirculo = 0;
		int numDardos = 50000000;
		para (int i = 0; i <numDardos; i ++) {
			// obtener un x aleatorio, y en un 2x2
			double x = doubleAleatorio (-1, 1);
			double y = doubleAleatorio (-1, 1);
			// prueba si el dardo aterrizo en el circulo
			double dist = Math.sqrt(x * x + y * y);
			if (dist <= 1) {
				numEnCirculo ++;
			}
		}
		// Si no lanzamos al doble, esto siempre será 0.
		double fracción = (doble) numEnCirculo / numDardos;
		// fraccion = pi / 4;
		double pi = fracción * 4;
		imprimir(pi);
	}
}
