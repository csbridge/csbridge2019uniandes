Castillos de arena

Folletos: Consola, Generador Aleatorio
Ejemplos trabajados: E = MC2, Fibbonacci, Carbono

Escriba los siguientes tres programas para empezar a aprender las variables con Java.

1. Hola mundo

Escriba un programa de consola que imprima Hola mundo (o cualquier mensaje que desee)

2. Peso de la luna

El usuario (un terrícola) ingresa su peso en la tierra. Tu programa imprimirá el peso del usuario en la luna, que es el 16.5% de su peso en la tierra.

3. Números aleatorios

Escriba un programa que imprima 1,000 números al azar entre 0 y 100.

El archivo NumerosAleatorio.java tiene el código para generar un número aleatorio. 

public class NumerosAleatorio extends EsConsole {

   public void run () {
      // cambie este código para imprimir 1000 números al azar
      // en el rango de 0 a 100.
      int ejemplo = intAleatorio(0, 100);
      imprimir(ejemplo);  
   }
}
