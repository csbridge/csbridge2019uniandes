Generador aleatorio

Escrito por Danielle Kain y Lisa Yan

Los objetos aleatorios te permiten agregar azar a tus programas. Puedes crear varios tipos de objetos aleatorios usando los métodos que te ofrece EsConsole. Por ejemplo:
int enteroAleatorio = intAleatorio(); //Devuelve un número entero aleatorio
double doubleAlAzar = doubleAleatorio(); //Devuelve un double aleatorio
Color colorAlAzar = colorAleatorio(); //Devuelve un color al azar
boolean booleanAlAzar = booleanAleatorio(); //Devuelve un booleano aleatorio

Enteros

La llamada al método sin parámetros devuelve un entero aleatorio sin límites, es decir, puede ser un número inmenso o uno muy pequeño. Es posible definir un rango para los números enteros generados al pasarlo como parámetro a la función.

int minimoEntero = 0;
int maximoEntero = 10;
intAleatorio(minimoEntero, maximoEntero); // Número entero en el rango [0,10]

Doubles

La llamada al método por defecto, es decir, sin parámetros, devuelve un double entre 0.0 y 1.0. Para aumentar el rango de los doubles generados, también: podemos especificar límites.

double y = doubleAleatorio(-3, 5.5); // Nota: -3 se convierte de entero a double (-3.0)

Booleanos

A diferencia de los enteros y los doubles, hay un solo parámetro p que define el comportamiento de booleanAleatorio(). El número que se ingresa al método afecta la probabilidad con la que el boolean adquiera un valor de true o false.

  • Devuelve true con probabilidad p.
  • Devuelve false con probabilidad 1-p.
  • p debe ser un doble entre 0 y 1.0.
  • El valor predeterminado de p es 0.5.
boolean probabilidadIgual = booleanAleatorio();
boolean probabilidadDesigual = rgen.siguienteBooleano(0.75); // devuelve verdad 3/4 del tiempo
