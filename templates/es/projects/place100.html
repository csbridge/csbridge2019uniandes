Coloque 100 (con solución)

Problema escrito por Chris

Folletos: KarelReferencia

Tu objetivo es hacer que Karel cree una pila de 100 beepers en la plaza enfrente de donde comienza. ¡¡¡Por favor!!! ¡No escribas ponerBeeper 100 veces seguidas! (pista: usar un ciclo for.)

Solución 

/ **
 * Programa : Place100
 * ------------------
 * Este programa hace que karel coloque un montón de
 * 100 beepers. Buenos tiempos
 * /
La clase pública Place100 extiende SuperKarel {
	
	public void run () {
		moverse ();
		// Este for-loop repetirá el código dentro de 100 veces.
		para (int i = 0; i <100; i ++) {
			ponerBeeper ();
		}
		moverse ();
	}
}
