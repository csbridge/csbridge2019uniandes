Que sigue

Creado por Kerem Goksel y Chris Piech

A pesar de que tuvimos un intenso período de dos semanas en el que cubrimos muchos temas fundamentales en Informática, esta es solo una pequeña parte de lo que hay que aprender en este apasionante campo. El primer escalón que tienes después de sentirte cómodo con el material de este curso es terminar los temas de CS 106A, la clase introductoria original en la Universidad de Stanford en la que se basa esta clase.

Acabado CS 106A

CS 106A es una clase intensa de 10 semanas en Stanford, cubre algunos temas más que nosotros y tiene otros proyectos que no tuvimos tiempo de cubrir en este curso. Sin embargo, hicimos evaluaciones que cubren los conceptos que omitimos para que puedas aprender los conceptos restantes. Para comenzar, debes instalar la versión de Eclipse de Stanford en tu propia computadora, como se describe en este folleto de instalación de Eclipse.

La Línea (Revisión)

Primero, hicimos una tarea que revisa los conceptos que cubrimos en clase. Descarga el código de inicio de la página y completa la línea. Esta tarea solo revisa los temas que ya hemos cubierto en clase, por lo que no necesitas aprender ningún tema nuevo para hacerlo.

Ahorcado (cadenas)

Como saben, las cadenas (String) son un tipo de variable que usamos para almacenar secuencias de caracteres en la memoria de la computadora, y generalmente las usamos a lo largo del curso para realizar un seguimiento de cosas como mensajes informativos o puntuaciones que se muestran en los juegos. Sin embargo, ¡hay mucho más acerca de las cadenas que eso! De hecho, muchos problemas interesantes en Informática implican trabajar con cadenas. Si bien no hemos aprendido mucho acerca de lo que puedes hacer con las cadenas en este curso, una excelente manera de comenzar a aprender es crear un proyecto que requiera que trabajes con cadenas. La tarea que hacen todos los estudiantes de Stanford para aprender cadenas es construir el juego de Ahorcado ("Adam asmaca", como lo llaman los turcos). Continuamos y escribimos un folleto especial para el ahorcado para ti y agregamos el código de inicio en el sitio web del curso. Puedes ir allí, leer el folleto, descargar el código a tu computadora, importar el proyecto a Eclipse y comenzar a trabajar en él inmediatamente.

Instakilo (HashMaps)

Un último concepto que tienes que aprender es HashMaps. Revisa el concepto de HashMaps en la documentación, y luego intenta el problema de Instakilo.

Despues de CS 106A

Una vez que hayas terminado con el material de CS 106A, puedes continuar y explorar cualquier campo de las ciencias computacionales que desees. Puedes acceder al material en CS106B, el segundo curso de introducción al plan de estudios de Stanford, pero hay una gran cantidad de recursos en línea que enseñan diferentes conceptos de CS de forma gratuita, por lo que puedes elegir lo que deseas aprender y hacerlo a tu propio ritmo, desde tu propia computadora. Hemos creado esta guía práctica para algunos de los buenos cursos en línea que son apropiados para tu nivel, pero sitios web como Khan Academy, Codecademy, Udemy y Coursera tienen muchos otros cursos que también puedes seguir, si decide profundizar aún más. .

Cursos de Programación Web

Para practicar:

Cubrió parte del material en estos cursos, pero tomarlos puede ser una buena idea si desea actualizar y aplicar sus conocimientos antes de continuar.

  • Codecademy HTML y CSS: este curso enseña los conceptos básicos de HTML y CSS desde cero. Puede ser una buena introducción al desarrollo web.
  • Codecademy JavaScript: este curso enseña la sintaxis básica de JavaScript y va un poco más lejos de lo que lo hicimos. Buen punto de partida si quieres hacer más con JavaScript.

Sobre la base de JS básico, HTML, CSS

  • Codecademy jQuery: enseña a usar jQuery (un práctico framework de JavaScript) para crear páginas web interactivas.
  • Coursera: Internet, Historia, Tecnología y Seguridad: No es un curso de programación, pero enseña sobre la historia de Internet y, en general, cómo funciona. Tiene alguna información interesante sobre tecnologías de internet y seguridad web.
  • La aplicación Codecademy Make a Rails: lleva a construir una aplicación web básica desde cero utilizando Ruby on Rails, un framework web popular (tiene como prerequisito HTML básico, CSS, JS). Bueno para aquellos que desean un gran desafío después de JavaScript y para aquellos que desean crear aplicaciones web más complejas.

Gráficos

  • Khan Academy Advanced Javascript - Juegos y visualizaciones: puede combinar su conocimiento de JavaScript con su conocimiento gráfico para hacer juegos y otras aplicaciones gráficas geniales que se ejecutan en el navegador en este curso.
  • Gráficos 3D interactivos de Udacity: un curso de gráficos JS un poco más avanzado, puede ir después de KhanAcademy (el sitio web del curso no requiere experiencia previa, pero es útil tener conocimientos de JavaScript). Tiene aproximadamente 48 horas de material.

Programación Java

  • Introducción a la programación en Java de Udacity: aproximadamente 96 horas de material. Enseña Java desde cero (sin necesidad de programación previa), pero probablemente va más allá de lo que enseñamos. Puede usar este curso si desea profundizar en Java y aprender a programar Java fuera de las bibliotecas de Stanford.
  • Udacity Cómo hacer aplicaciones de Android: aproximadamente 60 horas de material. Enseña Android asumiendo una experiencia en Java decente. Esto puede ser bastante desafiante y complicado, por lo que es posible que desees tomar primero el otro curso de Java.

Otros temas interesantes de informática

  • Khan Academy Advanced JavaScript - Simulaciones naturales: aplicación de JS y programación básica para hacer simulaciones geniales + introducción a conceptos como la aleatoriedad.
  • Khan Academy Algorithms: Enseña implementaciones de algoritmos básicos en JS. Puede ser bueno después de todas las otras cosas de JS.

Recursos útiles

De ahora en adelante, continuará su viaje de aprendizaje en Ciencias de la Computación por su cuenta, ¡pero eso no significa que no haya recursos para ayudar! Si bien no podemos estar allí para revisar su código y enseñarle nuevos conceptos, puede usar el foro de clase que creamos en Piazza para ayudarse mutuamente. Si necesita una referencia completa de todos los métodos que utilizamos, puede encontrar la documentación de ACM aquí y la documentación general de Java aquí. Más adelante, una vez que haya terminado con los temas restantes en CS106A y desee moverse a nuevos temas, un sitio web llamado Stack Overflow será su mejor amigo. Es un sitio web de preguntas y respuestas utilizado por muchos programadores de todo el mundo. Si tiene un problema, es probable que alguien más haya tenido ese problema antes y probablemente haya formulado esa pregunta en Stack Overflow. Simplemente, al buscar su problema en este sitio web probablemente le dará ideas sobre cómo abordar un problema, ¡al igual que las sugerencias de los Líderes de la Sección!

Una nota sobre el uso del código de otros

Ver el código de calidad escrito por programadores experimentados puede ser muy útil para aprender, ya que le mostrará formas elegantes e inteligentes de resolver problemas y formas adecuadas de formatear su código, por lo que no hay nada de malo en mirar el código de otras personas cuando está aprendiendo. Sin embargo, como la programación de aprendizaje se convierte en su viaje personal, tenga en cuenta que esto es algo que está haciendo por sí mismo, ¡porque lo ama! Copiar el código de otra persona puede proporcionarle una gratificación instantánea al permitirle resolver un problema rápidamente, pero perjudicará su aprendizaje y no obtendrá la satisfacción de crear una solución a un problema, que es uno de los momentos más bellos de la programación. No hay nadie por ahí para comprobar si copia el código de otros como en la escuela, pero recuerde que hacerlo mata toda la diversión de la programación.
