Referencia de Eventos

Written by Danielle Kain, Kevin Miller, and Nathan Dalal

Listeners de Mouse

Cuando usas eventos de mouse, debes escribir el método agregarMouseListeners en la primera línea del método run(). Esto indica al programa que debe reaccionar a los eventos de MouseEvento.
agregarMouseListeners();
A continuación se encuentran todos los métodos que puedes usar para describir el movimiento y uso del mouse en tu código. Recuerda que los métodos no se llaman manualmente sino que se invocan cuando el evento sucede.
public void mouseMovido(MouseEvento e) // cuando muevas mouse
public void mouseArrastrado(MouseEvento e) // cuando arrastres mouse
public void mousePulsado(MouseEvento e) // cuando pulses en mouse
public void mouseSoltado(MouseEvento e) // despues de pulsar cuando el mouse solte
public void mouseClickeado(MouseEvento e) // click (pulsado y soltado en misma ubicación)
public void mouseSale(MouseEvento e) // cuando mouse sale del canvas
public void mouseEntra(MouseEvento e) // cuando mouse entra el canvas
Esto es un ejemplo de escribir uno de esos métodos. 
public void mousePulsado(MouseEvento e) {
	imprimir("X: " + e.darX() + "y:" + e.darY());
	//Esto imprimirá las coordenadas de donde presionas mouse.
}

Listeners de teclas

Para escuchar al teclado, como los eventos del mouse, necesitas llamar a este método en run() para agregar los métodos de tecla.
agregarTeclaListeners();
Antes de funcionar tienes que hacer click al canvas.
public void teclaPulsada(KeyEvent e) // cuando pulses en cualquier tecla
public void teclaSoltada(KeyEvent e) // despues de pulsar cuando soltes de la tecla
public void teclaEspichada(KeyEvent e) // una escrita de cualquier tecla
Para usar los eventos de KeyEvent, tienes que importar esta librería. Ten cuidado porque esta librería, como puedes ver en el ejemplo abajo, es en inglés. Si necesitas ayuda con la lengua o con escuchar al teclado, pregunta a tu líder de sección.
import java.awt.event.KeyEvent;
Este es un ejemplo de escuchar por los movimientos de "WASD". 
public void teclaEspichada(KeyEvent e) {
	if (e.getKeyCode() == KeyEvent.VK_W) {
		imprimir("arriba");
	}
	if (e.getKeyCode() == KeyEvent.VK_A) {
		imprimir("izquierda");
	}
	if (e.getKeyCode() == KeyEvent.VK_S) {
		imprimir("abajo");
	}
	if (e.getKeyCode() == KeyEvent.VK_D) {
		imprimir("derecha");
	}
}
